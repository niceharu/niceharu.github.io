`Object.hasOwnProperty` 를 활용한다.

```js

let obj1 = {name: 'tony', age: '18'}

console.log(obj1.hasOwnProperty('name')) // true
console.log(obj1.hasOwnProperty('age')) // true
console.log(obj1.hasOwnProperty('phone')) // false
```
