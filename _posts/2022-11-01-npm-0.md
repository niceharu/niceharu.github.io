npm 모듈 설치시 아래와 같이 버전명을 입력하지 않으면 최신 버전이 설치된다.

```bash
$ npm install mongoose-auto-increment
 ```

dependency 문제로 특정 버전을 설치해야 하는 경우 `@version`을 붙여 설치한다.

```bash
$ npm install mongoose-auto-increment@3.2.0
 ```
