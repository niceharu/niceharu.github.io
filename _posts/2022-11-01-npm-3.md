## Package 전체 설치
```bash
yarn
 ```

## Package 개별 설치
```bash
yarn add [package]
yarn add [package]@[version] # 특정버전 설치
yarn add -D [package] # devDependencies에 추가
yarn global add [package] # global 추가
 ```

## Package 업데이트
```bash
yarn upgrade [package]
yarn upgrade [package]@[version]
 ```

## Package 삭제
```bash
yarn remove [package]
```

## Package 목록 표시
```bash
yarn list
```
