### 이미지 추출
```bash
docker save -o <추출 파일명> <추출할 docker image명>
docker save -o myserver.tar myserver:1.0
```

### 이미지 삽입
```bash
docker load -i <추출한 파일>
docker load -i myserver.tar
```
